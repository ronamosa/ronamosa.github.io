# 📦 Resources & Links

Starter projects, event documents, and external tools for the Tech for Good Hackathon 2026.

## 🚀 Starter Projects

Five pre-configured starter projects are available — one per example theme. Use the **Git repo** if students can clone from GitHub on the day, or the **ZIP** if you need to copy it onto devices ahead of time.

| Project | Theme | Git Repo | ZIP Download |
|---|---|---|---|
| 🌊 **Water Watch NZ** | Ocean / Water Pollution | [github.com/ronamosa/ocean-pollution-starter](https://github.com/ronamosa/ocean-pollution-starter) | [Download ZIP](https://github.com/ronamosa/ocean-pollution-starter/archive/refs/heads/main.zip) |
| 💛 **How Are You Today?** | Youth Wellbeing | [github.com/ronamosa/youth-wellbeing-starter](https://github.com/ronamosa/youth-wellbeing-starter) | [Download ZIP](https://github.com/ronamosa/youth-wellbeing-starter/archive/refs/heads/main.zip) |
| 🍲 **Kai Share** | Community Food Sharing | [github.com/ronamosa/community-food-sharing-starter](https://github.com/ronamosa/community-food-sharing-starter) | [Download ZIP](https://github.com/ronamosa/community-food-sharing-starter/archive/refs/heads/main.zip) |
| 🦎 **Kaitiaki Watch** | Native Species Conservation | [github.com/ronamosa/native-species-starter](https://github.com/ronamosa/native-species-starter) | [Download ZIP](https://github.com/ronamosa/native-species-starter/archive/refs/heads/main.zip) |
| ♻️ **Green School Tracker** | School Sustainability | [github.com/ronamosa/school-sustainability-starter](https://github.com/ronamosa/school-sustainability-starter) | [Download ZIP](https://github.com/ronamosa/school-sustainability-starter/archive/refs/heads/main.zip) |

:::
