Stream Deck XL Linux Setup - Complete Configuration Guide for Ubuntu

My quick setup notes for getting streamdeck XL going on Linux.

Steps​

  1. Follow "Streamdeck UI documentation" to get things setup on your desktop.
  2. Setup --user level systemd service for streamdeck so it runs in the background (read: "Create A Systemd Service")

Files​

~/.config/systemd/user/streamdeck.service

[Unit]
Description=StreamDeck UI
DefaultDependencies=no

[Service]
Type=simple
ExecStart=/home/rxhackk/.local/bin/streamdeck
Restart=on-failure
RestartSec=10

[Install]
WantedBy=default.target

load and enable

# as user
systemctl --user daemon-reload
systemctl --user enable streamdeck.service
systemctl --user start streamdeck.service
