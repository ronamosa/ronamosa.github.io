Brother MFC-J4330DW Printer Setup on Linux - Complete Driver Installation Guide

Setting up my Brother MFC-J5330DW Network Printer on Ubuntu.

Tested and works on: Ubuntu, VERSION="22.04.1 LTS (Jammy Jellyfish)"

Add Network Printer‚Äã

Bring up your printer menu (I use start and type "printer" Gnome shell).

Add a printer

Type your printers IP address in the box and then select your printer.

I'm going for the whole MFC-J5330DW multi-function device.

Brother Printer Driver‚Äã

Go to the Brother website, Downloads page.

Select Linux, (for me) Linux (deb), and OK (or just go here for the shortcut to download).
